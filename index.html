TeraStitcher

A tool for fast automatic 3D-stitching of teravoxel-sized images
(BMC Bioinformatics 2012, 13:316)

Download this project as a .zip file Download this project as a tar.gz file

TeraStitcher is a free tool that enables the stitching of Teravoxel-sized tiled microscopy images even on workstations with relatively limited resources of memory (<8 GB) and processing power. It exploits the knowledge of approximate tile positions and uses ad-hoc strategies and algorithms designed for such very large datasets. The produced images can be saved into a multiresolution representation to be efficiently visualized (see Vaa3D-TeraFly) and processed.

For more info, please visit our GitHub Wiki.
